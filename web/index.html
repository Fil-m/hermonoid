Hermonoid

Веб-чат для Hermes Agent на Android. Працює через Termux. Жодного root — просто термінал і браузер.

⚡ One‑Click Install 📱 Android + Termux 🔗 Веб‑інтерфейс 🧠 Hermes Agent
1

Встанови Termux

Завантаж з F-Droid або GitHub — там найсвіжіша версія.

⬇ Завантажити Termux (F-Droid) ⬇ GitHub Releases
2

Встав і виконай команду

Відкрий Termux, скопіюй і встав цю команду та натисни Enter:

bash <(curl -s https://raw.githubusercontent.com/Fil-m/hermonoid/main/install.sh)
⏱ Встановлення займає 1–3 хвилини. Все робиться автоматично — оновлення, залежності, клонування, запуск.
3

Перейди в браузер

Посилання http://localhost:8080 скопіюється в буфер автоматично. Встав його в будь-якому браузері на телефоні — і ти в системі.

✅

Готово! Починай спілкування

У веб-інтерфейсі налаштуй підключення до моделі, обери або створи сесію — і спілкуйся з Hermes Agent в реальному часі.
